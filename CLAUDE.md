# Therapist WebSite

This is a web site of a therapist Vanessa Aders. The website is built using React, Typescript, Webpack. The source code is at
https://github.com/alexmt/moksha-mermaid. The site is hosted by Github pages and available at https://iamthat.love/.

# Development

Use `pnpm start` to start development server. The server will be available at http://localhost:1234.

When working on tasks check open http://localhost:1234 to see the changes in real time. Assess the changes in the browser and make sure they
are correct. Iterate until changes are matching the requirements.
